多层向前神经网络

组成部分

  • 输入层
  • 隐藏层
  • 输出层

    概念

  • 每层由单元组成
  • 输入层是由训练集的实例特征向量传入
  • 经过连接点的权重传入下一层，每一层的输出是下一层的输入
  • 隐藏层的个数可以是任意的 输入层 输出层都只有一层
  • 每个单元称作神经节点
  • 以上称为2层神经网络（输入层不算）
  • 一层中的加权求和，然后根据非线性方程转化输出
  • 作为多层向前神经网络 理论上有足够多的隐藏层和足够大的训练集可以模拟出任何方程

    设计神经网络结构

    1、使用神经网络训练数据之前，必须确定神经网络的层数，以及每层单元的个数
    2、特征向量在被传入输入层时通常被先标准化到0和1之间（为了加速学习过程）
    3、离散型变量可以被编码成每一个输入单元对应一个特征值可能赋的值
    比如：特征值A可能取三个值（a0,a1,a2)，可以用三个输入单元来代表A
    如果A=a0 那么代表a0的单元取1 其余取0

神经网络既可以解决分类问题，也可以解决回归问题

  • 对于分类问题 如果是两类 我们可以用一个输出单元表示如果多于两类 每一个类用一个输出单元表示 所以输出层的单元数量通常等于类别的数量
  • 没有明确的规则设计最好有多少个隐藏层 根据准确度要求

Backpropagation算法 反向传播算法

  • 通过迭代性来处理训练集的实例
  • 对比经过神经网络后输入层的预测值与真实值之间
  • 反方向（从输出层=>隐藏层=>输入层）来以最小化误差更新每个连接的权重
    算法介绍
    D：数据集 ，l 学习率，一个多层向前神经网络
    一个训练好的神经网络（权重优化好了）
    权重乘以值再加上偏向 每个节点有个非线性转化方程 再转化 就是该节点的输出



    （l）学习率 相当于蜗牛迈步的大小 步子小太慢 步子大有可能跨过 Errj 当前层的误差 乘以oj当前层的值
    权重更新:当前位置+更新量

    终止条件

  • 权重的更新低于某个阀值
  • 预测的错误率低于某个阀值
  • 达到预设一定的循环次数
